milsymbol.js - 0.6

Milsymbol is a library in pure javascript that creates SVG symbols according to MIL-STD-2525C, MIL-STD-2525D and NATO STANAG APP6(b). Since version 0.5.5 native canvas output is supported as well as SVG.

All unit symbols with SIDCs in both 2525 and APP6 are supported. The symbols are created according to APP6(c)/MIL-STD-2525D symbology and might therefore differ from the symbols in shown in the standard documents, but they are made according to the standards and follows the way that symbols are created in ADRP 1-02 (US Army Doctrine Reference Publication).

Milsymbol is just a single file, and you just have to load it once to get access to all symbols available.

The API documentation will show you everything about how milsymbol is used.

The library is published under a MIT-license so you can do basically anything you want with it as long as you keep the copyright notice.

Please let me know if you have used milsymbol in any public projects, it would be nice to have a usage gallery here.

Milsymbol is drawn according to standards

Implemented standards

This library implements several different standards, and you can select if you want it to draw according MIL-STD-2525 or NATO APP6, but every SIDC will always return a symbol, so even if you specify that you want symbols according to MIL-STD-2525 and give milsymbol a SIDC that only exists in APP6, you will get the correct symbol.
The following pages tries to display most of the tables and figures in the different standards, using milsymbol to draw all symbols so that you can compare milsymbol with the standard documents.

  • MIL-STD-2525D, implemented in milsymbol.

Flexibility

milsymbol is designed to be very flexible, and you can change most things about the symbols using the API, like selecting any size, to adding your own color schemas, and if you want to you can even inject custom code to display additional information.
All symbols will provide you with information about the anchor point for the center of the octagon, and the anchor point for the correct placement. This makes it easy to use milsymbol symbols as markers on a map. And milsymbol won't just draw the symbol, it will also place all texts around your symbol and store the information in a structured way.
Please read the API documentation to get information about all possiblities in milsymbol.

About

All of the code started out as a proof of concept and was initially just thought as a simple way for me to get symbols for some webmapping demos I was making as a hobby, but the code expanded into something much more. The development is still a hobby project coded during late nights and weekends by Måns Beckman.
